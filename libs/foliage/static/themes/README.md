# Theme Thumbnails

This directory contains 400x300px PNG thumbnails for all 10 curated themes.

## Naming Convention

- `{theme-id}-thumb.png`
- Example: `grove-thumb.png`, `night-garden-thumb.png`

## Themes

1. grove
2. minimal
3. night-garden
4. zine
5. moodboard
6. typewriter
7. solarpunk
8. cozy-cabin
9. ocean
10. wildflower

## Specifications

- **Dimensions**: 400x300px
- **Format**: PNG
- **Target file size**: <50KB each
- **Generation**: Programmatically generated by rendering ThemePreview.svelte to canvas

## Usage

These thumbnails are referenced in the theme definitions at `src/lib/themes/registry.ts`:

```typescript
thumbnail: "/themes/{theme-id}-thumb.png";
```

The thumbnails are served from the `/static/` directory, which SvelteKit automatically serves at the root path.
