## Example Completed Journey

```markdown
# Naming Journey: Code Reviewer

## What Is This Thing?

**Fundamentally:** A creature that examines code carefully and leaves thoughtful comments.

**What does it DO?**
Reviews pull requests: finds issues, acknowledges good patterns, provides actionable feedback.

**What emotion should it evoke?**
Sharp but fair. Observant. A little intimidating but ultimately helpful.

## Walking Through the Forest

I enter the grove. Bloodhound tracks through underbrush, Elephant builds steadily,
Lynx repairs PR feedback already given...

I need something that *gives* code review feedback rather than receives it.
Something intelligent, detail-oriented, that notices what others miss.

I walk past the predators ‚Äî Panther strikes too fast, too focused.
I need a thinker.

## Candidates

**Crow** üê¶‚Äç‚¨õ
- Natural meaning: Highly intelligent bird, known for problem-solving and memory
- Why it fits: Crows recognize patterns, remember faces, leave "gifts" (good feedback)
- The vibe: Smart, slightly unnerving, ultimately helpful
- Potential issues: Could feel negative if "crow" evokes death

**Hawk** ü¶Ö
- Natural meaning: Sharp-eyed predator, sees detail from above
- Why it fits: Survey/overview view, catches what others miss
- Potential issues: Already used (hawk-survey for security)

**Magpie** üê¶
- Natural meaning: Collects shiny things, very clever
- Why it fits: Collects issues like shiny things
- Potential issues: Too playful for serious code review

## Selection: Crow

Crow's intelligence and pattern recognition fits perfectly. The "leaving gifts"
behavior ‚Äî crows leave objects for humans who feed them ‚Äî maps to providing
good feedback. And the slightly unnerving-but-helpful vibe is right.

**Workflow name:** The Review

**Phases:**
```
PERCH ‚Üí OBSERVE ‚Üí CAW ‚Üí GIFT ‚Üí FLY
```
```

## The Three Questions (Always Answer These)

1. **What is it?** (fundamentally, in nature)
2. **What does it do?** (in the user's workflow)
3. **What emotion should it evoke?** (the feeling it creates)
