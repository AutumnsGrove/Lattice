# Grove Domain Search - Comprehensive Research Mission

**Mission:** Find the perfect base domain for a multi-tenant blog platform where users get subdomains like `username.YOURDOMAIN.com`

**Context:** You have access to all Grove project documentation and research artifacts. Review them to understand the platform's vision, technical architecture, and business model before beginning the search.

---

## PROJECT ESSENCE (Quick Reference)

**What it is:** Multi-tenant blog platform on Cloudflare Pages with SvelteKit. Users get their own blogs on subdomains. Features a shared community "feed" where blogs can opt-in to share posts, vote, and react with emojis.

**Core metaphor:** A grove of trees - blogs are individual trees that grow together in a shared forest. When users share to the feed, they're creating a connected ecosystem.

**Vibe:** Druidic gathering place. Casual, organic, cozy, inviting. Light playfulness but with freedom. Easy to approach. Think: campfire circle in a forest clearing, not corporate boardroom.
