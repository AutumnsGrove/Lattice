## What IS This Thing?

### Fundamentally, it is:
- Not a place (unlike Meadow, Nook, Clearing)
- Not a structural feature (unlike Lattice, Heartwood)
- Not a tool you use (unlike Wisp, Forage)
- More like... atmosphere. Ambiance. Texture.

### What does it DO?
- Adds warmth without demanding attention
- Creates subtle visual interest
- Makes spaces feel alive, handmade
- Provides decorative backgrounds
- Transforms images into character art
- Works with Glass UI to add depth

### What emotion should it evoke?
- Warmth (like sunlight through leaves)
- Whimsy (like fireflies at dusk)
- Calm (like watching clouds drift)
- Comfort (like familiar textures)
