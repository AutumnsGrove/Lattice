## What IS a help center?

Let me sit with this.

A help center is:
- Documentation (knowledge written down)
- Answers to questions (searchable)
- Contextual hints (little "?" buttons that appear where you need them)
- A reference (you go back to it repeatedly)
- Self-service (you find the answer yourself)

It's NOT:
- A conversation (that's Porch)
- A status update (that's Clearing)
- A roadmap (that's Trails)

## But a help center is different...

When I use a help center, I:
- **Search** for what I need (active query)
- **Read** detailed explanations (not just direction)
- **Click** contextual help buttons (it appears WHERE I'm stuck)
- **Return** to it repeatedly (reference material)

A waystone doesn't answer questions.
A waystone doesn't appear when you're confused.
A waystone doesn't know WHAT you need.

**A waystone is a sign. A help center is a guide.**
